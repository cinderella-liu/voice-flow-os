# 上传与构建

1. 在 GitHub 仓库首页点击 **Add file → Upload files**。
2. 上传本项目文件夹内的全部内容（不要只上传 ZIP）。
3. 提交说明填写：`Initial Voice Flow OS v0.1.1`。
4. 打开 **Actions → Android CI → Run workflow**。
5. 构建成功后，在运行详情页底部下载 `voice-flow-os-debug-apk`。
6. 解压后安装 `app-debug.apk`。

首次测试：打开 App，允许麦克风和通知；锁屏后点亮屏幕，点击 Voice Flow OS 通知并说“日历，测试日程”。
