# Voice Flow OS

Voice Flow OS 是一个以语音为入口的 Android 个人工作流中枢。

## 当前版本：0.1.0（锁屏语音入口 MVP）

已实现：

- Activity 可显示在锁屏之上。
- 常驻通知提供锁屏入口，点击后自动开始识别。
- 使用 Android `SpeechRecognizer` 将中文语音转成文字。
- 保存原始指令到本地日志。
- “日历/开会”类指令可打开系统日历新增草稿。
- 为滴答清单、IMA 与 AI 路由预留命令分发层。

尚未实现：

- 自然语言时间解析。
- 自动写入滴答清单。
- 自动写入 IMA。
- 云端或本地 LLM 意图解析。
- OPPO/ColorOS 专项后台保活适配。

## 构建

1. 使用最新版 Android Studio 打开仓库根目录。
2. 安装 Android SDK 35 与 JDK 17。
3. 等待 Gradle Sync。
4. 连接 OPPO 手机，开启开发者选项与 USB 调试。
5. 运行 `app`。

首次启动时允许：

- 麦克风
- 通知
- 锁屏通知显示（ColorOS 设置中）

建议在 ColorOS 中进一步允许：自启动、后台运行、锁屏通知，并关闭本应用的电池优化。

## 首次验收

1. 启动应用并授权。
2. 锁屏。
3. 点亮屏幕，点击 Voice Flow OS 通知。
4. 不输入密码，确认应用出现在锁屏上并开始聆听。
5. 说：“日历，测试 Voice Flow OS”。
6. 检查是否打开 OPPO 日历新增事件页面。
